Meu Portfólio - Landing Page

 Sobre o Projeto

Este projeto consiste em uma landing page de portfólio desenvolvida com o objetivo de apresentar informações pessoais, habilidades técnicas, projetos e exemplos de integração com APIs. O site foi criado para servir como uma apresentação profissional, reunindo em um único local informações relevantes sobre conhecimentos e experiências na área de desenvolvimento web.

 Desenvolvimento

O projeto foi desenvolvido utilizando HTML para a estrutura da página, CSS para estilização e responsividade, e JavaScript para adicionar interatividade e integração com serviços externos. Durante o desenvolvimento foram aplicados conceitos de organização de conteúdo, navegação entre seções, responsividade para diferentes tamanhos de tela e consumo de APIs utilizando requisições assíncronas com a função `fetch()`.

 Funcionalidades

* Apresentação pessoal em formato de landing page.
* Seção de habilidades e conhecimentos.
* Navegação entre seções da página.
* Links para redes profissionais (GitHub, LinkedIn e Instagram).
* Integração com APIs da BrasilAPI.
* Consulta de CEP com exibição de endereço.
* Consulta de DDD com exibição das cidades correspondentes.
* Consulta de CNPJ com informações da empresa.
* Consulta de bancos através do código bancário.

 Objetivo Acadêmico

Este projeto foi desenvolvido como atividade da disciplina de Front-End, ministrada pelo professor William Hubner. Seu principal objetivo foi aplicar na prática os conhecimentos adquiridos durante as aulas, incluindo estruturação de páginas web, estilização com CSS e integração com APIs utilizando JavaScript.
