Meu Portfólio

Este projeto foi desenvolvido como atividade da disciplina de Front-End, ministrada pelo professor William Hubner. O objetivo foi criar uma landing page de portfólio para apresentar informações pessoais, habilidades e projetos desenvolvidos. Para sua construção foram utilizados HTML, CSS e JavaScript, além de integrações com APIs da BrasilAPI para consultas de CEP, DDD, CNPJ e bancos. O projeto também conta com links para redes profissionais, como GitHub, LinkedIn e Instagram, demonstrando conhecimentos em desenvolvimento web e consumo de APIs.
